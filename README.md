## Overview

This repository contains the full code for my MSc thesis on deepfake detection using CNN-based models. The project trains and evaluates two models — **EfficientNet-B4** and **XceptionNet** — on a large-scale balanced dataset of real and StyleGAN-generated faces, and analyses detection performance across demographic subgroups (gender and age).

**Research Questions:**
- How effectively can CNN-based deepfake detectors identify StyleGAN-generated faces?
- Do EfficientNet-B4 and XceptionNet exhibit disparate detection performance across demographic subgroups?
- What patterns can be identified through Grad-CAM visualisation and error analysis to explain model failures?

---

## Dataset

**140k Real and Fake Faces** (Kaggle)  
https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces

- 70,000 real faces (FFHQ + CelebA)
- 70,000 fake faces (StyleGAN-generated)
- Split: 70% train / 15% validation / 15% test

After downloading and unzipping, place the dataset at:
```
data/real_vs_fake/real-vs-fake/{train,valid,test}/{real,fake}/*.jpg
```

## Usage

### Train both models (requires GPU)
```bash
python3 scripts/thesis_train.py
```
This trains EfficientNet-B4 and XceptionNet sequentially, saves the best checkpoints, and runs demographic bias analysis. Results are saved to `./results/`.

On a SLURM cluster:
```bash
sbatch --job-name=train \
  --output=results/train.out \
  --wrap="python3 scripts/thesis_train.py"
```

### Generate ROC curves and confusion matrices
```bash
python3 scripts/roc_auc_check.py
```
Requires `results/predictions_cache.csv` (generated during training).

### Run demographic bias analysis
```bash
python3 scripts/demographic_analysis.py
```

### Evaluate on UTKFace (out-of-distribution real faces)
```bash
python3 scripts/both_models_utk.py
```
Requires UTKFace dataset at `data/utk/UTKFace/`.

### Generate Grad-CAM heatmaps
```bash
python3 scripts/gradcam_analysis.py
```

## Results Summary

| Model | Accuracy | F1 | AUC-ROC | False Negatives |
|---|---|---|---|---|
| EfficientNet-B4 | 99.90% | 99.90% | 0.9999 | 5 / 10,000 |
| XceptionNet | 71.72% | 60.57% | 0.9840 | 5,656 / 10,000 |

**Key finding:** XceptionNet exhibited a strong prediction bias toward classifying all faces as real, resulting in a 56.56% false negative rate. EfficientNet-B4 demonstrated near-perfect detection with minimal demographic bias.
